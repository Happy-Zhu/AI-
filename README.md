# AI 扫盲小白课程

这是一套面向 AI 小白用户的系统扫盲课程，目标是帮助没有技术基础的人理解 AI 的基本原理、常见名词、工具选择方法和真实使用边界。

## 当前状态

- GitHub 仓库：<https://github.com/Happy-Zhu/AI-.git>
- 当前课程规划控制在 15 节以内。
- 第 01-10 课已完成视频口播稿和飞书文档版草稿。
- 第 11-15 课仍处于未开始状态。
- 第 01-03 课已制作统一风格 SVG 示意图，素材统一放在 `assets/` 目录。

## 主要文件

- `AI扫盲小白课程_项目档案.md`：项目目标、写作规范、协作规则和阶段边界。
- `AI扫盲小白课程_汇总索引.md`：课程目录、每节课状态、文件位置、素材需求和修改记录。
- `AI扫盲小白课程_目录优化版.md`：当前优化后的课程目录。
- `视觉素材风格预设.md`：课程配图、示意图和流程图的统一视觉风格要求。
- `assets/`：课程示意图素材。
