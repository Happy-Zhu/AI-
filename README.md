# AI 扫盲小白课程

这是一套面向 AI 小白用户的系统扫盲课程，目标是帮助没有技术基础的人理解 AI 的基本原理、常见名词、工具选择方法和真实使用边界。

## 当前状态

- GitHub 仓库：<https://github.com/Happy-Zhu/AI-.git>
- 当前课程规划控制在 15 期以内。
- 第 01-10 期已完成视频口播稿和飞书文档版草稿。
- 第 11-15 期仍处于未开始状态。
- 第 01-03 期已制作统一风格 SVG 示意图，素材统一放在 `assets/` 目录。
