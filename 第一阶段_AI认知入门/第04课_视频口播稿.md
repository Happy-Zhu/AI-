# 第 04 课视频口播稿：AI 工具怎么选？文本、图片、视频、办公工具一次讲明白

## 基本信息

- **建议时长**：8-10 分钟
- **主讲开场**：大家好，我是嘿皮
- **课程基调**：真诚、不废话、有对比、有判断
- **本节目标**：让小白建立工具选择方法，而不是收藏一堆工具名

## 口播正文

### 1. 开场：工具不是奖杯

大家好，我是嘿皮。

前面三节课，我们讲了 AI 的基本逻辑，也讲了模型、Token、上下文、Prompt、API 这些词。

这一节课，我们讲一个特别现实的问题：AI 工具到底怎么选？

很多小白刚开始学 AI，最容易陷入工具清单。

今天收藏 100 个工具，明天看 10 个榜单，后天又注册一堆账号。

最后收藏夹很满，但真正能稳定解决问题的工具没几个。

所以这节课先给结论：

小白选 AI 工具，不要从“哪个最强”开始，要从“我要解决什么问题”开始。

工具不是奖杯，不是越多越好。工具是入口，关键是任务。

### 2. 先分 5 类

对小白来说，AI 工具先分 5 类就够了。

第一类，文本对话类。

比如 ChatGPT、Claude、DeepSeek、Kimi、豆包。它们主要帮你解释概念、总结资料、写作、分析、翻译。

第二类，搜索问答类。

比如 Perplexity、秘塔、ChatGPT 搜索、Kimi 搜索。它们更适合查资料、找来源、追热点。

第三类，图像生成类。

比如 ChatGPT 图像、即梦、Midjourney、通义万相。它们适合做封面、配图、视觉灵感。

第四类，视频生成类。

比如即梦、可灵、Runway、Pika、Sora。它们适合做分镜、短片段和创意草稿。

第五类，办公与执行类。

比如飞书、Notion、Coze、Dify、Codex。它们开始进入文档、知识库、工作流和任务执行。

你先有这张地图，再看工具就不会乱。

### 3. 文本类工具怎么选？

文本类是小白最应该先学的。

截至 2026 年 5 月 14 日，我给你一个简单判断。

ChatGPT 适合作为主线工具。综合能力强，生态完整，能做对话、写作、总结、图片、文件、代码和 Codex 工作流。OpenAI API 文档里 GPT-5.1 标注 400K context，但 ChatGPT 网页端具体额度要看套餐。

Claude 适合长文档、逻辑分析、代码和写作润色。Claude 付费聊天常见 500K context，部分 API 或长上下文能力可到 1M，但国内访问和订阅门槛更高。

DeepSeek 适合中文问答、推理和性价比 API。这里要注意区分网页端和 API 端，DeepSeek 官方 API 价格页当前列出的 `deepseek-chat` 是 64K context；如果网页或 App 有灰度长上下文能力，也要以你实际使用入口为准。

Kimi 适合长文档阅读和资料整理。它一直以长上下文出名，目前官方模型列表里 K2 预览模型是 128K context，但模型版本变化快，具体能力要看官方开放平台。

豆包适合国内轻量日常使用，访问方便，功能入口多，但专业深度和稳定性要按任务测试。

你不用背这些。

先记一句：主线用 ChatGPT，国内备 DeepSeek/Kimi，长文档可试 Claude/Kimi，轻量日常可用豆包。

### 4. 真实案例：写一篇公众号文章

假设你要写一篇公众号文章，主题是“普通人为什么要学 AI”。

你不要只问哪个工具最强。

你要把任务拆开。

找选题角度，可以用 ChatGPT 或 DeepSeek。

查资料和案例，可以用 ChatGPT 搜索、Kimi 或秘塔。

整理长资料，可以用 Kimi、Claude 或 ChatGPT。

写提纲和初稿，可以用 ChatGPT 或 Claude。

最后改成你的口吻，还可以继续用 ChatGPT 或 Claude。

一个任务可以拆成多个环节，每个环节选更合适的工具。

这才叫工具选择。

### 5. 五个判断标准

以后你看到任何 AI 工具，都问 5 个问题。

第一，它解决什么具体问题？

第二，它比我现在的主力工具强在哪里？

第三，它有没有访问、账号、付费、网络门槛？

第四，它的输出能不能稳定复现？

第五，它会不会带来隐私、版权、商用风险？

如果这 5 个问题答不上来，不要急着充值。

先用免费额度或小额测试，跑一个真实任务。

### 6. 结尾总结

这一节课总结一下。

小白选 AI 工具，先按任务分类，不要被工具清单带跑。

文本类优先学 ChatGPT，国内备 DeepSeek、Kimi、豆包，长文档和写作可以试 Claude/Kimi。

真正的问题不是哪个工具最强，而是：我现在要解决什么问题，哪个工具能用最低成本给我稳定结果。

课后任务：

列出你现在最想用 AI 做的一件事，然后写清楚：任务是什么、属于哪类工具、候选工具是什么、用什么结果判断好坏。
