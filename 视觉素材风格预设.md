# AI 扫盲小白课程视觉素材风格预设

## 1. 统一视觉方向

本课程的视觉素材统一采用“干净克制的飞书知识库课程图解风”。

核心目标不是炫技，而是帮助小白理解抽象概念。

适合用于：

- 概念解释图；
- 流程示意图；
- 对比图；
- 小白理解类比图；
- 视频录制时的辅助画面；
- 飞书文档中的插图。

## 2. 中文提示词预设

后续调用 GPT 作图、AI 生图或手工设计图片时，优先使用以下提示词作为基础模板：

```text
请生成一张适合飞书云文档课程使用的中文信息图，主题是【替换为本图主题】。

视觉风格：干净、克制、现代、知识库教程风，类似轻量 SaaS 产品说明图，不要夸张科幻风，不要赛博朋克，不要复杂背景。

画面要求：
1. 使用浅灰白背景，整体留白充足；
2. 主色使用深绿色/青绿色作为强调色，辅色使用蓝色和少量暖黄色；
3. 采用扁平化矢量插画和信息图结构；
4. 画面中可以有简单图标、流程箭头、卡片式模块，但不要过度装饰；
5. 中文文字要清晰，字号大，适合放在飞书文档里阅读；
6. 信息层级明确，最多 3-5 个核心模块；
7. 不要出现真实品牌 Logo，不要出现人物照片，不要出现密集小字；
8. 整体观感要专业、轻松、适合 AI 小白课程。

输出比例：16:9。
```

## 3. 英文提示词预设

如果某些工具更适合英文提示词，可以使用：

```text
Create a clean modern Chinese infographic for a Feishu/Lark knowledge-base course. Topic: [replace with topic].

Style: minimal, professional, educational, light SaaS documentation style. Use a soft off-white background, deep teal as the primary accent color, blue as a secondary color, and a small amount of warm yellow for emphasis. Flat vector illustration, simple iconography, clear flow arrows, modular cards, generous whitespace.

Requirements:
- 16:9 aspect ratio.
- Chinese text must be large and readable.
- Use only 3 to 5 key modules.
- No cyberpunk style, no sci-fi glow, no complex background, no real brand logos, no photorealistic people.
- The image should help complete beginners understand the concept quickly.
```

## 4. 课程图片使用规范

- 一节课建议 1-3 张核心视觉素材，不要为了“丰富”而堆图。
- 抽象概念优先做示意图，例如 Token、上下文、推理机、幻觉、Agent。
- 流程类内容优先做流程图，例如“输入 -> AI 处理 -> 人类判断 -> 输出”。
- 对比类内容优先做表格或双栏图，例如“搜索引擎 vs AI”。
- 图片中的文字要少，解释放在正文里。
- 所有图片都要服务理解，不做纯装饰图。

## 5. 当前课程固定视觉元素

- 背景色：浅灰白、低饱和白色。
- 主强调色：深绿色/青绿色。
- 辅助色：蓝色、少量暖黄色。
- 字体气质：清晰、现代、适合中文阅读。
- 构图：卡片、箭头、双栏对比、三步流程。
- 禁止：赛博朋克、复杂科技光效、过暗背景、密集数据图、品牌 Logo 堆叠。
